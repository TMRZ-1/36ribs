# 36ribs
Here you can find everything you need to make your own 36ribs keyboard. It's my first project, so it's simple and it works. I can confirm that it's version 1.0.

BOM for pcb
1. 1x pro micro rp2040 controller
2. 36x 1n4148 or 1n4148w diodes, for better looks take through hole diodes
Optional. 36x hotswap sockets

BOM for case

2. 16x m2 4x3.2 inserts
3. 16x m2 5mm screws

BOM for acrylic cover
1. acrylic cover, I've made 3mm for my build
2. 8x m2 10mm standoffs
3. 16x m2 5mm screws
